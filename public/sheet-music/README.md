# Starter Violin Library - Sheet Music Files

This directory contains PDF files for the Starter Violin Library.

## Required Files:

1. `mary-had-a-little-lamb.pdf`
2. `hot-cross-buns.pdf`
3. `twinkle-twinkle-little-star.pdf`
4. `lightly-row.pdf`
5. `go-tell-aunt-rhody.pdf`
6. `ode-to-joy.pdf`
7. `long-long-ago.pdf`
8. `minuet-in-g.pdf`
9. `allegretto.pdf`
10. `song-of-the-wind.pdf`

## Adding Files:

Place public-domain PDF sheet music files here with the exact filenames listed above.

The app will automatically load these files when users select songs from the Explore page.

## Note:

If PDFs are missing, the app will show a clear error message indicating which file is expected.
